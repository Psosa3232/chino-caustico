# Chino Cáustico

**Chino Cáustico** es una herramienta de estudio interactiva diseñada para dominar el vocabulario del nivel **HSK 1**. Esta aplicación transforma el aprendizaje de caracteres chinos en una experiencia visual y táctil, ideal para su uso en tablets y dispositivos móviles.

## Características actuales
* **Flashcards dinámicas:** Sistema de cartas con Hanzi, Pinyin y traducción al español.
* **Filtrado por categorías:** Estudia específicamente sustantivos, verbos, adjetivos o pronombres.
* **Interfaz Adaptativa:** Diseño responsive optimizado para pantallas táctiles.
* **Modo PWA:** Instalable en la pantalla de inicio de tu tablet para acceso rápido sin barras de navegación.

## Tecnologías utilizadas
* **HTML5 / CSS3:** Estructura y diseño con animaciones 3D para las cartas.
* **JavaScript (Vanilla):** Lógica del sistema de filtrado y navegación entre secciones.
* **Manifest & Service Workers:** Configuración para funcionamiento como Progressive Web App.
